# jobs/

One folder per job application. Each application folder contains the JD and all `/draft-cv` run outputs.

**Structure:**
```
jobs/[company_slug]-[role_slug]/
  YYYY-MM-DD_HH-MM_jd.md       ← save the JD here manually
  YYYY-MM-DD_HH-MM/             ← one subfolder per /draft-cv run
    analysis.md
    draft-cv.yaml
    html-cv/cv(harvard).html    ← added by ./bin/render-cv ... --theme harvard
    draft-letter/
      draft-letter.yaml         ← added by /draft-letter
      html-letter/
        letter(modern).html     ← added by ./bin/render-letter ... --theme modern
```

This folder is generated by `/draft-cv`. Do not edit files here manually.
