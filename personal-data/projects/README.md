# projects/

One file per project or role. These are the raw building blocks `/draft-cv` scores and
selects from when tailoring your CV to a job description.

**Naming:** `[company_slug]-[project_name].md` — Example: `tnt_lab-dashboard_redesign.md`

**Schema:** See `agents-ref/schema.md` for required fields, tag taxonomy, and stack naming.

**Getting started:** Run `/personal-log` — it will guide you through adding your first project.
Want to see a populated example? Check out the [`examples`](../../tree/examples/personal-data/projects) branch.
