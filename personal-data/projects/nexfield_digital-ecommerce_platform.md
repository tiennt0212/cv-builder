## Description
Multi-vendor ecommerce marketplace for a retail client, supporting 40+ vendor storefronts with product listings, cart, and order management. Frontend team of three, working cross-timezone with the client's PM.

## Achievements
- Improved Largest Contentful Paint (LCP) by 38% — from 5.1s to 3.2s — by implementing route-based code splitting, lazy-loading product image grids, and memoising expensive GraphQL query results.
- Built a reusable cart and checkout flow consumed across 40+ vendor storefronts — designed with a composable component model that reduced per-storefront integration effort from 3 days to under 4 hours.
- Led GraphQL schema review sessions with the backend team before sprint start, catching two breaking field changes early and preventing a mid-sprint API renegotiation.

## Valued Inputs
- Proposed migrating from REST to GraphQL for product listing queries after prototyping both approaches — reduced over-fetching and improved mobile load performance.
- Created a shared component storybook for the marketplace UI kit — adopted as the handover artefact for the client's internal dev team after project completion.
- Acted as cross-timezone liaison between the studio FE team and the client PM — structured async summaries after each sprint review to close the communication gap.
