# companies/

One file per employer. Each file stores company context used by `/draft-cv` to populate
employment dates and company descriptions.

**Naming:** `[company_slug].md` — lowercase, underscores between words. Example: `tnt_lab.md`

**Schema:** See `agents-ref/schema.md` for required fields and enum values.

**Getting started:** Run `/personal-log` — it will guide you through adding your first employer.
